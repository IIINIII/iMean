> Welcome to iMean <

iMean is a NOSQL injectable app to help you learn about NOSQL Injection. Click here to find more about NOSQL Injection. iMean consists of 3 login pages each using different method for sending data to server. These methods are:
  1. JSON data with POST method (Secure 1 and Insecure 1)
  2. Simple form POST method (Secure 2 and Insecure 2)
  3. GET method (Secure 3 and Insecure 3)
You may need Burp Suite to change some requests. If there is no users try registering one.
